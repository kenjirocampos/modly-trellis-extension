Aquí tienes una guía detallada para crear el esqueleto de tu extensión en **Windows usando Linux (a través de WSL, Windows Subsystem for Linux)**.

### Requisitos Previos en tu Windows
1.  **NVIDIA GPU:** Esencial para ejecutar estos modelos de IA localmente.
2.  **WSL 2:** Si no lo tienes, abre PowerShell como administrador y ejecuta `wsl --install`. Instalará Ubuntu por defecto.
3.  **VS Code:** Con la extensión oficial **"WSL"** para editar el código de Linux cómodamente desde Windows.
